# GitOpsTerraformAzureK8sPipelinePoc
This project is a poc to showcase Go/Java langugaue microservice deployed to Azure via Github Actions CI/CD pipeline with JFrog as the artifactory and Azure managed Postgres DB using Terraform and ArgoCD running on Azure K8s using Azure vault and toher resources.

## This is the root app which binds together following 3 repos in this same Org: local-platform-infra + springboot-crud-service + go-crud-service
